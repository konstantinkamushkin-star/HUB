# Импорт дайвсайтов из базы данных

## Описание

Скрипт `import_dive_sites_worldwide.js` импортирует 4000+ реальных дайвсайтов из разных регионов мира в базу данных.

## Подготовка

1. Убедитесь, что база данных PostgreSQL запущена и доступна
2. Убедитесь, что PostGIS расширение установлено в базе данных
3. Проверьте переменные окружения или настройте подключение к базе данных в скрипте

## Переменные окружения (опционально)

```bash
export DB_HOST=localhost
export DB_PORT=5432
export DB_DATABASE=divehub
export DB_USERNAME=admin
export DB_PASSWORD=admin123
```

## Запуск

```bash
cd backend
node import_dive_sites_worldwide.js
```

## Что делает скрипт

1. Генерирует 4000+ дайвсайтов на основе известных локаций по всему миру:
   - Красное море (Египет, Судан, Эритрея, Джибути, Саудовская Аравия)
   - Мальдивы
   - Индонезия (Бали, Комодо, Раджа Ампат)
   - Филиппины
   - Карибское море
   - Большой Барьерный Риф (Австралия)
   - Галапагосы (Эквадор)
   - Остров Кокос (Коста-Рика)
   - Палау
   - Микронезия (Чуук, Яп)
   - И многие другие регионы

2. Проверяет существующие дайвсайты в базе данных
3. Импортирует только новые дайвсайты (пропускает дубликаты)
4. Обновляет координаты latitude/longitude из PostGIS location

## Результат

После успешного выполнения скрипта в базе данных будет минимум 4000 уникальных дайвсайтов с:
- Названиями
- Координатами (PostGIS geography)
- Типами сайтов (reef, wall, wreck, cave, drift, shore, boat, other)
- Уровнями сложности (1-4)
- Глубинами (min/max)
- Морской жизнью
- Странами и регионами

## Примечания

- Скрипт использует батчи по 50 сайтов для оптимизации производительности
- При ошибках в батче скрипт пытается импортировать сайты по одному
- Дубликаты определяются по комбинации имени и координат
