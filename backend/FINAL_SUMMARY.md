## Проблемы и ограничения

1. **Overpass API (OpenStreetMap)**
   - Медленный или недоступный
   - Возвращает мало данных
   - Решение: использовать региональные запросы, но это все равно медленно

2. **Недостаточно данных для 2000+ дайвсайтов**
   - Публичные источники ограничены
   - Коммерческие базы данных не имеют открытого API
   - Web-скрейпинг нарушает ToS

3. **Требуется больше реальных дайвсайтов**
   - Нужно добавить еще ~1400 дайвсайтов
   - Источники: известные дайв-гиды, PADI/SSI базы, краудсорсинг
