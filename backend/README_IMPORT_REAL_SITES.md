## Для добавления 2000+ реальных дайвсайтов

Необходимо расширить список реальных дайвсайтов в файле `import_real_dive_sites_only.js`.

### Источники реальных дайвсайтов:
1. PADI - официальная база дайв-центров
2. DiveAdvisor - каталог дайвсайтов
3. ScubaEarth - база данных дайвсайтов
4. OpenStreetMap - геоданные о дайв-центрах
5. Известные дайв-гиды и путеводители

## Запуск импорта

```bash
node import_real_dive_sites_only.js
```

Скрипт автоматически:
- Проверяет существующие дайвсайты
- Импортирует только новые
- Избегает дубликатов
