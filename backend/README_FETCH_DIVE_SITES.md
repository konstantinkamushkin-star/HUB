# Загрузка реальных дайвсайтов из различных источников

## Проблема с Overpass API

Overpass API часто медленный или недоступный. Поэтому используем комбинированный подход:

1. **Известные дайвсайты** - ручной список проверенных реальных дайвсайтов
2. **OpenStreetMap** - когда API доступен (опционально)
3. **Другие источники** - ReefBase, NOAA (в разработке)

## Текущие скрипты

### 1. `fetch_dive_sites_comprehensive.js`
Создает базовый список известных дайвсайтов. Сейчас содержит ~58 дайвсайтов из Красного моря.

**Использование:**
```bash
node fetch_dive_sites_comprehensive.js
```

**Результат:** `dive_sites_comprehensive.json`

### 2. `fetch_dive_sites_osm.js`
Пытается загрузить дайвсайты из OpenStreetMap через Overpass API.

**Использование:**
```bash
# Пропустить глобальный запрос (рекомендуется)
node fetch_dive_sites_osm.js --skip-global
```

**Проблема:** API часто таймаутится или медленный.

### 3. `fetch_dive_sites_osm_fast.js`
Быстрая версия с упрощенными запросами.

**Использование:**
```bash
node fetch_dive_sites_osm_fast.js
```

## План действий для получения 2000+ дайвсайтов

### Шаг 1: Расширить список известных дайвсайтов

Отредактируйте `fetch_dive_sites_comprehensive.js` и добавьте больше дайвсайтов в массив `KNOWN_DIVE_SITES`.

**Источники для расширения:**
- Существующий файл `import_dive_sites_worldwide.js` (уже содержит ~224 дайвсайта)
- Известные дайв-гиды и путеводители
- PADI, SSI базы данных (если доступны)

### Шаг 2: Объединить существующие данные

```bash
# Использовать существующий список из import_dive_sites_worldwide.js
node -e "
const existing = require('./import_dive_sites_worldwide.js');
const diveSitesData = existing.diveSitesData || [];
const fs = require('fs');
fs.writeFileSync('dive_sites_from_existing.json', JSON.stringify(diveSitesData, null, 2));
console.log('Сохранено', diveSitesData.length, 'дайвсайтов');
"
```

### Шаг 3: Импортировать в базу данных

Создать скрипт импорта, который:
1. Читает JSON файлы с дайвсайтами
2. Проверяет дубликаты в базе
3. Импортирует только новые дайвсайты
