# Модуль поездок - Итоговая документация

## ✅ Что было реализовано

### 1. Модели данных (Models)

#### Trip.swift
Полная модель поездки со всеми необходимыми полями:
- Тип поездки (daily/safari)
- Выбор отеля (для daily) или яхты (для safari)
- Детализация цен по номерам/каютам (ныряющие/не ныряющие)
- Страна, даты поездки
- Минимальный уровень сертификации и количество погружений
- Фотографии и описание
- Количество мест с расчетом свободных мест
- Участники поездки
- Доступные курсы
- Наличие нитрокса
- Лидер группы (для дайвцентров)
- Программа поездки (по дням с детализацией)
- Дополнительные расходы
- Доступность аренды снаряжения

#### Course.swift
Модель курса с:
- Уровнем курса (basic/advanced/professional/technical/specialization)
- Описанием
- Системами обучения (PADI, SSI, NAUI и т.д.)
- Программой курса (модули)

#### Расширение Instructor
Добавлены поля:
- photoURL (главное фото)
- description (подробное описание)
- trainingSystems (системы обучения)
- credentials (регалии и достижения)

#### Hotel.swift и Yacht.swift
Модели для отелей и яхт с типами номеров/кают

### 2. ViewModel

#### TripViewModel.swift
- Загрузка списка поездок с фильтрацией
- Загрузка отелей, яхт, курсов, инструкторов
- Создание, обновление, удаление поездок
- Бронирование поездок
- Проверка прав на создание поездки (дайвцентр или PRO подписка)

### 3. Представления (Views)

#### TripsListView.swift
- Список всех поездок
- Фильтрация по типу, стране, датам, нитроксу, аренде снаряжения
- Детальный просмотр поездки
- Кнопка создания поездки (для авторизованных пользователей)

#### CreateTripView.swift
Полная форма создания поездки с:
- Выбором типа поездки
- Выбором отеля/яхты
- Заполнением всех обязательных полей
- Загрузкой фотографий
- Настройкой программы по дням
- Добавлением дополнительных расходов
- Настройкой цен
- Выбором курсов
- Выбором лидера группы (для дайвцентров)

#### TripBookingView.swift
Экран бронирования с:
- Добавлением участников
- Выбором курсов
- Выбором аренды снаряжения
- Расчетом общей стоимости
- Подтверждением бронирования

### 4. API Integration

#### NetworkService.swift
Добавлены endpoints:
- `getTrips(filters:)` - получение списка поездок с фильтрацией
- `getTrip(id:)` - получение деталей поездки
- `createTrip(_:)` - создание поездки
- `updateTrip(_:)` - обновление поездки
- `deleteTrip(tripId:)` - удаление поездки
- `bookTrip(tripId:participants:)` - бронирование поездки
- `getHotels()` - получение списка отелей
- `getYachts()` - получение списка яхт
- `getCourses(diveCenterId:)` - получение курсов
- `getDiveCenterInstructors(diveCenterId:)` - получение инструкторов дайвцентра

### 5. Интеграция в приложение

Поездки интегрированы в SocialTabView как отдельная вкладка "Trips"

## 🚀 Как использовать

### Для пользователей:

1. Откройте вкладку "Social" → "Trips"
2. Просматривайте доступные поездки
3. Используйте фильтры для поиска нужной поездки
4. Нажмите на поездку для просмотра деталей
5. Нажмите "Book" для бронирования

### Для дайвцентров и PRO пользователей:

1. Откройте вкладку "Social" → "Trips"
2. Нажмите "+" для создания новой поездки
3. Заполните все необходимые поля
4. Сохраните поездку

## 📝 Примечания

- Все даты должны быть в формате ISO 8601
- Фотографии должны загружаться через отдельный endpoint
- Цены должны иметь валюту
- При бронировании проверяется соответствие сертификации требованиям
- Количество мест автоматически обновляется при бронировании
